PlanetScale for PostgreSQL and Digital Ocean both required credit card to even make a free account. So I couldn't do that.
UI can be made better and more user friendly if required.

To run locally:
1. Go to the github repo: https://github.com/pravin-as/Recruiter-Tool
2. Click on green code button and download zip file. Extract the zip file.
3. Open the folder in a terminal and do "cd recruiter-tool-backend". Now run "npm install". Then "npm start". 
4. Now do "cd .." for coming back to root folder. Then do "cd recruiter-tool-frontend". Now run "npm install". Then "npm start".
5. Now open http://localhost:3000/ in any browser.
